# alim596.github.io

My CV, formatted as a component datasheet — because every engineer has read one,
and no recruiter has ever received one.

Live at **https://alim596.github.io**. It also prints properly; try it.

Built with [Astro](https://astro.build) (zero client JS except pin-diagram
hover and a copy button), IBM Plex, and one signal-red accent. Deployed to
GitHub Pages on every push.

```bash
npm install
npm run dev     # localhost:5180
```

[Quorum](https://github.com/alim596/quorum) · [Argus](https://github.com/alim596/argus)
